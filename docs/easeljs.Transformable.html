Class: Transformable

Transformable

new Transformable()

implements all transformations (move, scale, rotate)
can exclude individual transformations by unsetting:

  • this._moves = false
  • this._rotates = false
  • this._scales = false

to false before calling parent constructor

  • Parent.call(this);
