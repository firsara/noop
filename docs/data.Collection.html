Class: Collection

data. Collection

new Collection(model, data)

A collection holds and manages several models in an array

Parameters:
Name Type Description
model String

the class name of the collection's model (i.e. user.hasMany = 'apps' //model = 'apps')
data object | Number

is either a data object or an id of the model that should be fetched
Source:
  • data/Collection.js, line 19
Example
var collection = new Collection('User', [1, 2, 10]);
collection.pull(_pulled);
