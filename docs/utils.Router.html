Class: Router

Router

new Router(controller, routes)

handles routes defined as javascript object and maps them to specific controllers and their according action

Parameters:
Name Type Description
controller Object

an object that gets passed on to every controller when route changes are detected
routes Object

a list of routes, see example for configuration
Source:
  • utils/Router.js, line 10
Example
var controller = {layout: ...};
var router = new Router(controller, {...routes})
router.parse(controller.layout);
router.route();
